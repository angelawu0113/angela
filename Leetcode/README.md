# 學習筆記
題目：709 To Lower Case <br>
目的：將所有輸入的字串都轉為英文小寫輸出<br>
str.upper()          #把所有字符中的小寫字母轉換成大寫字母<br>
str.lower()          #把所有字符中的大寫字母轉換成小寫字母<br>
str.capitalize()     #把第一個字母轉換為大寫字母，其餘小寫<br>
str.title()          #把每個單字的第一個字母轉換為大写，其餘小寫<br>
